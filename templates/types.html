COMPUTER TYPES



Type Name Definetion
Desktop computer A desktop computer is a computer that fits on or under a desk.
Laptop a portable computer that is more efficient,
and nearly as powerful, as a desktop computer.
Netebook portable computer about half the size of a traditional laptop
Hybrid A 2-in-1 PC comprises a screen and a detachable keyboard
Tablet a mobile computing device designed to be held in one or two hands
Smartphone a mini computer that can browse the Internet
and run software programs like a computer
